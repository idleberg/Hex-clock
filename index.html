Hex o'clock

About

Based upon hex clock, a clock using the hex-value of the current time as background color, and my own fl_Ed interface experimentation. This fork lets you manipulate the color and text parameters seen on this website. All changes will be sabed in your browser's localStorage and restored on future visits.
